# 🏋️ Gym Power - Fitness & Training Platform

A responsive single-page website for a fitness and training platform, built with a focus on clean UI, mobile-first design, and smooth user experience.

🔗 **Live Demo:** [https://ahmed1-dev.github.io/Gym-Power-Project/](https://ahmed1-dev.github.io/Gym-Power-Project/)

## 📋 Overview

Gym Power is a single-page application (SPA) that showcases a fitness club's training levels, weekly workout schedules, class timings, and membership plans — all in one smooth-scrolling page.

## ✨ Features

- **Responsive Design** – Fully mobile-first layout that adapts across all screen sizes.
- **Level Selection** – Interactive Beginner / Intermediate / Advanced workout plans, each with a detailed 3-day weekly split.
- **Class Schedule** – A weekly timetable showing course names and available time slots.
- **Membership Plans** – Pricing cards for monthly, semi-annual, and annual memberships.
- **Optimized Assets** – Compressed images (WebP) and structured layout for faster load times.

## 🛠️ Built With

- **HTML5** – Semantic markup
- **CSS3** – Flexbox, Grid, Media Queries, Animations
- **Bootstrap** – Grid system & responsive components
- **JavaScript** – DOM manipulation for interactive sections
